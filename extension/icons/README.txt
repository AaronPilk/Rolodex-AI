Drop in the icons here:

- icon16.png  (16x16, browser toolbar)
- icon48.png  (48x48, extensions page)
- icon128.png (128x128, install dialog + Chrome Web Store)

Use the same Apple-blue R wordmark from the rest of the Rolodex brand.
Quick way: take the SVG from the brand-mark in app.html, render at the three sizes
in any image tool (Sketch, Figma, Affinity, even macOS Preview can do PNG export).

If these are missing Chrome will fall back to a generic puzzle-piece icon —
the extension still works, it just looks generic.
